PhpLivePdo::__construct

Creates a PDO instance representing a connection to a database

 

Sintaxe (PHP >= 8)

public PhpLivePdo::__construct ( string $Ip, string $User, string $Pwd, string $Db [, string $Drive = 'mysql' [, string $Prefix = '' [, string $Charset = 'utf8' [, int $TimeOut = 5 ] ] ] ] );

 

Parameters

$Ip - Database address, ip or dns;

$User - User to authenticate;

$Pwd - Password to authenticate;

$Db - Default database schema;

$Drive - The database used;

$Prefix - Text to be concated with "_" and table name;

$Charset - The character set;

$TimeOut - Maximun seconds to run a query;

 

Sintaxe (PHP < 8)

public PhpLivePdo::__construct ( array $Options );

 

Parameters

$Options:

Drive (string)(Optional - Default: mysql) - The database used;

Ip (string) - Database address, ip or dns;

Db (string) - Default database schema;

Charset (string)(Optional - Default: utf8) - The character set;

User (string) - User to authenticate;

Pwd (string) - Password to authenticate;

TimeOut (int)(Optional- Default: 5) - Maximun seconds to run a query;

Prefix (string)(Optional) - Text to be concated with "_" and table name;

 

Return Values

Returns a PDO object on success.

 

Errors

Throws a PDOException if the attempt to connect to the requested database fails.

 

Dependencies

- PHP >= 7.4;

- One of the PDO PHP extensions;

 

Examples

// PHP >= 8
$PDO = new PhpLivePdo( 'localhost', 'root', 'password', 'mysite', Prefix:'mydb' );

 

$PDO = new PhpLivePdo([
'Ip' => 'localhost',
'User' => 'root',
'Pwd' => 'password',
'Db' => 'mysite'
]);
