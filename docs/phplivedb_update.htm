PhpLiveDb::Update

Create an object to run Update querys

 

Sintaxe

namespace ProtocolLive\PhpLiveDb;

public function PhpLiveDb::Update(
string $Table
): Update;

 

Parameters

$Table - The table to perform the update. The table alias can be specified

 

Return Values

An object to add options and run the update.

 

Examples

$consult = $Db->Update('users u');
