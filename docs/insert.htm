PhpLiveDb::Insert

Create an object to run Insert querys

 

Sintaxe

public ProtocolLive\PhpLiveDb\Insert(
string $Table
):ProtocolLive\PhpLiveDb\Insert;

 

Parameters

$Table - The table to perform the insert. The table alias can be specified

 

Return Values

An object to add options and run the Insert.

 

Examples

$consult = $Db->Insert('users u');
